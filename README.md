# rhetrica.css

> A text-oriented CSS for text documents.

Have a look at the demo: <https://nolze.github.io/rhetorica-css/>

## Features

* [x] Headings
  * [ ] Secondary super- and sub-texts
* [x] Paragraph
* [x] Sidenotes (in place of footnotes)
* [ ] Blockquote
* [ ] Epigraphs
* [ ] Images
* [ ] Tables
* [ ] Lists
* [ ] Table of Contents

## Principles

* Code less
* Don't be a framework
